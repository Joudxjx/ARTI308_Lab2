# ARTI308_Lab2
this repository is for my machine learning course

Dataset description: The dataset consists of metadata for approximately 8,560 movies. It includes features such as popularity, vote average (rating), and vote count, as well as title, overview, and release date.

Problem type: Regression

Variable: Vote average (rating)

Problem statement: The goal is to build a machine learning model that can predict a movie's rating based on its popularity and the number of votes it receives.
